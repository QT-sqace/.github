## 주요 기능
- 음성 회의 자동 요약
- 팀생성, 팀원 초대 및 삭제 기능
- 실시간 채팅 기능
- TODO 관리 및 JIRA 푸시
- 팀원 접속 상태 확인

## 기술 스택
### FrontEnd
- React, javascript, zustand
### BackEnd
- spring boot, java, jpa, jwt, websocket, feign client, fast api
### Infra
- Oracle Cloud, Kubernetes, Jenkins, Nginx, Elk stack, mysql, redis, Minio, Kafka
